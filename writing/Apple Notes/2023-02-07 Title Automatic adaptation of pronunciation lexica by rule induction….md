This project combines three of the projects proposed in the earlier presentation: Dialectal adaptation of pronunciation dictionaries, Word boundary adaptation for pronunciation dictionaries, and Sound changes within a language family.

The common core element of each of these projects is the use of phonetic rules, which are to be validated by data: in the former two cases, by using acoustic data to validate the results of the rule induction and/or application phases; in the latter, in the use of an external lexicon to show that there is an underlying common origin between the words.

The project should take an existing lexicon (or multiple), and through the use of phonological rules, extend or adapt the lexicon, to be fit for a purpose other than what it was originally designed for. This could be dialectal adaptation, adapting to word boundary effects, or even to adapting the pronunciation lexicon of one language to another.

The deliverables should include the software for rule induction and/or application, the resulting output lexicon, and an appropriate evaluation: if ASR is used, a lexicon-based system should be used, and the results should use word error rate; if TTS is selected, a listening test should be used.
